##### ADCs Specifications
| **Specification**  | **Symbol** | **Baseline requirement** | **Comment** | 
| ------------- | ------------- | ------------- | ------------- |
| Sampling rate  | fs  | ≥ 1.5MS/s | |
| Effective number of bits | ≥ 9 | Measured near Nyquist |
| Input Capacitance | Cin | ≤ 5pF | |
